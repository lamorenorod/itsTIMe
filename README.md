# itsTIMe
Unlearning challenge - TIM company

We used Collab, and we clones the original GitHub repo https://github.com/ashba93/hackathon-sapienza. So, you can clone the repositoryto verify the code

We used NegGrad+ (fine-tune on retain + gradient ascent on forget, combined) for the unlearning task.
